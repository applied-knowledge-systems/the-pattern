Covid19 - Redis Knowledge Graph in 3D

Explore medical literature graphically, leveraging industry knowledge and avoiding confirmation bias

Information exploration should be spatial and in VR

When exploring topics on science or engineering, you look at the diversity of the opinion, not the variety of the same cluster of words, same thought. You'd want to avoid confirmation bias and find articles relevant to the same concept, not necessary the ones which have similar words. This project "The Pattern" is the implementation of the following

Vision

Search or rather information exploration should be spatial preferably in VR (memory palace, see Theatre of Giulio Camillo).

VR/AR visualisation

Explore information in a spatial experience in virtual or augmented reality.

Force-directed graphs

A force-directed graph is a path towards it, where visual assisted by text — relevant text pops up on the connection and where people explore the concepts and then dig deeper into text.

Redis Ecosystem

Leveraging RedisLab modules, starting with Redis Gears, RedisGraph and RedisAI

Open Source

Build NLP pipeline leveraging Redis ecosystem whenever possible. We use COVID 19 (CORD19) medical articles corpus as input.

Implementation

Implementation

Redis components used: RedisGears, Redis Cluster, RedisGraph. Run all script

1. Ingest documents

Parses documents taking out body_text and saves under paragraphsin Redis cluster

2. Detect language

Uses Redis Gears by registering key reader on ‘paragraphs:*’.Not all medical literature was in English. Currently discards non-English: it’s “filter” operation

3. Split paragraphs into sentences

Map paragraphs into a sentence — flatmap operation

4. Spell checker

Uses symspell library and it’s “map” and outputs saved into hash — processor

5. Matcher

Uses pre-build Aho-Corasick automata to match incoming words to Medical Concepts using UMLS Methathesaurus

6. Populate RedisGraph

Loading matched sentences into edges in RedisGraph with Nodes being Concept (CUI)

7. Front-End repository

Stores code for this page and upcoming VR/AR

8. Flask API server to query RedisGraph

Just simple Flask API server
